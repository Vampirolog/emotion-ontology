Annotation Property: Format

Annotations (4)

  • Description "Typically, Format may include the media-type or dimensions of the resource. Format may be used to determine the software, hardware or other equipment needed to display or operate the resource. Examples of dimensions include size and duration. Recommended best practice is to select a value from a controlled vocabulary (for example, the list of Internet Media Types [MIME] defining computer media formats)." (en-us)
  • comment "The physical or digital manifestation of the resource." (en-us)
