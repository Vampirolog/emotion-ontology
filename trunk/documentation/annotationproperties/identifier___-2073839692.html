Annotations (4)

  • Description "Recommended best practice is to identify the resource by means of a string or number conforming to a formal identification system. Example formal identification systems include the Uniform Resource Identifier (URI) (including the Uniform Resource Locator (URL)), the Digital Object Identifier (DOI) and the International Standard Book Number (ISBN)." (en-us)
  • comment "An unambiguous reference to the resource within a given context." (en-us)
